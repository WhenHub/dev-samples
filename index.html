Developer Samples Home Page

Thanks for checking out the WhenHub API! These samples are here to help you see how to work with the API in real setting and to inspire your creativity. The samples are split up into different categories in case you want to see specific functionalities. Click the Display tab if you mainly want to see retrieving information and click the Edit tab if you're more intrested in posting data.

Many of the samples require that you get an Access Token from your WhenHub profile. If you're not sure where to get it, these directions will clear things up for you. While you're there, make sure you check out the Reference page, it will be helpful tool as you continue using the API.

Angular 4 - View on GitHub

This is a sample using Angular 4. The sample includes the following features:

  • List of schedules for user based on Access Token.
  • List of events per each Schedule.
  • Embedded Whencast for each Schedule.
  • Limited display of media objects for Events (images only).
  • Basic editing of title for schedules and events.
Edit and display data using Angular. The schedules you see are the ones that are in your WhenHub Profile.
